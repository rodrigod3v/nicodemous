# 🚀 nicodemouse v1.0.0 - Initial Release

Welcome to the first official release of **nicodemouse**! 🎉

This release brings seamless, cross-platform remote control for your devices, allowing you to share your mouse and keyboard across machines effortlessly via your local network.

## ✨ Key Features
- **Cross-Platform Support**: Run smoothly on both Windows (native Edge WebView2) and macOS (native WebKit).
- **Self-Contained Executables**: No external dependencies or complicated installers required! Just extract the ZIP and run.
- **Auto-Discovery**: Devices on your local network will find each other automatically.
- **Secure Pairing**: Connect instantly by comparing the PIN code shown on both machines.
- **Clipboard Syncing**: Seamlessly copy and paste text across your devices.
- **Responsive UI**: A modern, sleek web-based interface built with Vite and React.

## 📦 Downloads
- **Windows (x64)**: Download `nicodemouse-windows-x64.zip`, extract, and run `nicodemouse.exe`.
- **macOS (Universal)**: Download `nicodemouse-macos-universal.zip`, extract, and move `nicodemouse.app` to your Applications folder.

> ⚠️ **Note for macOS Users:**
> macOS requires explicit permission to monitor your mouse and keyboard. Please refer to our [Setup Guide](https://github.com/rodrigod3v/nicodemous/blob/main/MACOS_SETUP.md) for quick instructions on granting **Accessibility** and **Full Disk Access** permissions.

## 🐛 Known Issues & Feedback
As this is our first release, you might encounter some edge cases. If you find any bugs or have feature requests, please open an issue on our [GitHub repository](https://github.com/rodrigod3v/nicodemous).
